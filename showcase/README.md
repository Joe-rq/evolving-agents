# Showcase Assets

This directory keeps presentation and visual artifacts out of the engine's main
source tree.

## Contents

```text
demo/        # Standalone HTML demo pages used by examples and roadshow material.
poster/      # Poster images and poster HTML.
roadshow/    # Earlier roadshow deck and conversion helper.
roadshow_v2/ # Current roadshow deck, scripts, and operation notes.
projects/    # Generated slide-project workspaces and exported deck assets.
```

The reusable Python project remains in `core/`, `adapters/`, `experiments/`,
`examples/`, `content/`, and `docs/`. Files here are supporting materials for
explaining and presenting the project.
